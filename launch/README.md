# Launch kit — Sera FX

Ready-to-fire copy for the "spread the word" phase (see `../GROWTH.md` §5, §10).

**Fire order (don't skip):** ship → DNS live + HTTPS → verify indexed in Google + Bing →
API endpoints public → *then* launch. Launching to Hacker News with an unindexed site or
a 404 API wastes the one shot.

| File | Channel | When |
|---|---|---|
| `hacker-news.md` | Show HN | Tue–Thu, ~14:00–16:00 UTC, once API is live |
| `product-hunt.md` | Product Hunt | 00:01 PT on launch day |
| `x-thread.md` | X / @seraprotocol | Same morning, pin it |
| `reddit.md` | r/webdev, r/fintech, r/algotrading, r/SideProject | Space out; read each sub's rules first |

Angle that spreads: **free + open + machine-readable + AI-native**, funded by settlement
spread not data sales. Developers amplify "genuinely free, no catch." Lead with that.
